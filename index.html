Projetos de Desenvolvimento

Mememorável

Os memes estão presentes no nosso entretenimento desde a popularização da internet e, pensando em registrar e centralizar esse conteúdo para que ele seja facilmente acessado, desenvolvi o Mememorável. Nele o usuário pode navegar pelos últimos memes cadastrados, filtrá-los por ano, ordenar como desejar, realizar buscas dinâmicas, criar um perfil, salvar seus favoritos, denunciar publicações inadequadas e contribuir com a comunidade ao cadastrar novos memes. Outros desenvolvedores que desejarem ter acesso a esse banco de dados colaborativo podem consumir os dados através da API.

O processo de desenvolvimento foi ótimo para praticar minhas habilidades em PHP, Laravel, MySQL, HTML e CSS, estudar mais sobre JavaScript, requisições AJAX e criação e consumo de APIs. Também tive a oportunidade de conhecer conceitos novos para mim, como CORS, Regex, Tracking e o uso de SVGs.

UNIU

A UNIU nasceu com a missão de ser uma rede social para ambientes residenciais fechados. Hoje, os moradores acabam se organizando através de grupos de Whatsapp ou Facebook, que não são ideais para essa finalidade, porque todo mundo pode postar tudo o que quiser. Então, as informações importantes se perdem e a interação entre os moradores não acontece de uma forma saudável. Já no UNIU, os moradores podem interagir, reservar espaços, vender e trocar produtos, divulgar ações sociais, confirmar presença em eventos e receber avisos oficiais de forma segura e controlada por um moderador.

A plataforma foi desenvolvida como parte da formação em Desenvolvimento Web Full Stack pela Digital House através de um Projeto Integrador que explorou as tecnologias abordadas durante o curso. A aplicação foi construída utilizando o framework PHP Laravel através da arquitetura MVC, banco de dados relacional MySQL, JavaScript, JQuery, Materialize, CSS3 e HTML5. O desenvolvimento se pautou na metodologia ágil SCRUM, tendo sido dividido em 7 sprints ao longo de 5 meses de curso.

Samplaces

O Samplaces é uma plataforma para compartilhar fotos e histórias dos lugares favoritos em São Paulo para que possamos passear virtualmente em tempos de pandemia. Nele o usuário pode compartilhar seus lugares favoritos em categorias, clicando em "passear" ele pode encontrar fotos de outros usuários, explorar novos lugares na página "recentes" e, na página "top" encontrar os mais amados da cidade. Além disso, é possível se conectar com outros usuários e conhecer seus favoritos.

O site foi desenvolvido ainda na metade do curso de Web Full Stack da Digital House usando PHP nativo, MySQL, HTML e CSS. Durante o processo de desenvolvimento eu pude explorar o banco de dados e entender a comunicação com o PHP através do PDO.

PetShop

A plataforma foi parte do desafio de conclusão do módulo de PHP do curso de Web Full Stack da Digital House. Nele nós precisávamos criar um sistema CRUD que tivesse login, cadastro, edição e exclusão de usuários e produtos, além de exibir uma relação de todos os usuários e produtos cadastrados, como um usuário administrador de um e-commerce.

Para isso, decidi criar um CMS para um e-commerce de PetShop, utilizando PHP nativo, HTML, CSS e JSON como banco de dados.
